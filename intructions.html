• Create a git repository
  • Create SSH on your pc ( SEARCH IN INTERNET ) SSH-keygen ( chto to tipa togo )
  • Add ssh to your github account in settings-> SSH and GPG keys -> add ssh
  • Go to your working directory with terminal
  • type "git init" this initializes your git

  • In your working directory add a link to your git repository

    git remote add origin "(your ssh link to repository here) git@github.com:zhansdu/web_programming.git"

  • Next steps in your working directory

    add all your files to github repository

    git add .

    git commit -m"first commit"

    git push origin master

    git branch nurbek

    git checkout nurbek
